Selectores CSS los diferentes tipos y como funcionan. Los sub-artículos son los siguientes:

  • Selectores de tipo, clase e ID
  • Selectores de Atributos
  • Pseudo-clases y pseudo-elementos
  • Combinadores

Especificidad Tiene que ver en como los navegadores deciden qué valores de una propiedad CSS son más relevantes para un elemento y, por lo tanto, serán aplicados. InfoMoz...

Especificidad, es el método por el cual el navegador determina que estilos tienen mayor peso o relevancia de aplicación por sobre otros; el orden es así:

Elemento Nivel
Etiquetas 0
Clases, atributos y psudoclases 1
Selectores de id 2
